## LAVI's MATLAB OpenAI API connect Code

## Info
- 利用  MATLAB  撰寫程式，使其可自動連接 OpenAI API 並進行詢問，且將收到之回應輸出成文字檔，
並可將輸出內容自動以 Gmail 寄信給收件對象
